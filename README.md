# oop_fin_project - приложение управления личными финансами

**как пользоваться приложением**

Для того что бы начать пользоваться приложением необходимо зарегистрироваться и войти в свой аккаунт.
После этого вам станет доступны следующие возможности:

1 - добавить доход по определённой категории.  
2 - добавить расход по определённой категории.  
3 - установить бюджет в определённой категории.  
4 - просмотр статистики. Выводит всю доступную статистику по категориям.
